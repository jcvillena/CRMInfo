Before you build this project

Make sure that:

  • Your DevExpress.com account has access to DevExpress Blazor UI Components. This subscription is available as part of the Universal, DXperience or ASP.NET subscription.
  • Your DevExpress NuGet feed is registered as a package source.

Before you start this application

Refer to the supported browsers list. Additional polyfills are required for Internet Explorer (for example, you can create a Polyfill.io bundle to add promises).
